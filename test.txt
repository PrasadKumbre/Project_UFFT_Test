Git hub comments 

1. git init : Initialize a new Git repo.
2. git clone <repository URL>: clone a repo from a URL.
3. git add <file>: Add a file or changes to the stanging area.
4. git commit -m "<message>": Commit staged changes with a message.
5. git status: Check the status of your working tree.
6. git log: View commit history.
7. git branch: List, create, or delete branch.
8. git checkout <branch>: Switch to a different branch.
9. git marge <branch>: Marge changes from a different branch into the current branch.
10. git pull: Fetch and marge changes from a remote repository.
11. git push: Push local changes to a remote repository.
12. git remote add <name> <url>: Add a remote repository.
13. git fetch: Fetch changes from a remote repository.
14. git reset --hard HEAD: Reset the current branch to the last commit.
